
# 🐧 ldslockOS

**ldslockOS** is a Debian-based operating system focused 100% on Terminal productivity and extreme minimalism.

## 🚀 Overview
Unlike bloated operating systems, ldslockOS treats the terminal as the primary tool. You manage your apps, install packages, and control the system through lightweight windows, with a dynamic taskbar at the top.

## ✨ Features
* **Interface:** i3wm (Tiling Window Manager).
* **Terminal-First:** Terminal as the main app and command center.
* **Status Bar:** Real-time monitoring for Battery, Internet, and Time.
* **Lightweight:** Optimized for ultra-low RAM usage.
* **Debian Core:** Built on the stability and power of Debian.

## ⌨️ Keybindings
* `Win + Enter` : Open a new Terminal.
* `Win + C` : Open Control Center (Settings).
* `Win + D` : App search menu (Rofi).

## 🛠️ Installation
The system is currently built via GitHub Actions. You can download the ISO from the **Actions** tab once the build process is complete.
